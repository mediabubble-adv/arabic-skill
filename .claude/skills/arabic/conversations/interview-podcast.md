## 1. Arabic Podcast & Interview Landscape

### Market context (2024)
- Arabic podcast listening is growing 40%+ YoY; KSA and Egypt are the two largest markets
- **Top Arabic podcast categories:** Business/entrepreneurship, self-development, true crime, Islamic content, tech
- **Key platforms:** Spotify, Anghami, Apple Podcasts, YouTube (dominant for video)
- **Average Arabic podcast listener:** 25-40, urban, at least some college, commutes
- **Format preference:** Conversational over scripted; authentic "real talk" over polished corporate

### Interview culture in Arabic
- Guests are given more context before difficult questions — "cold" challenges feel rude
- Personal stories are expected and valued — Arab audiences connect through narrative
- Intellectual humility from the host ("سامحني لو السؤال غلط") before challenging questions is culturally appropriate
- Praising the guest at the start ("وسعيد جداً إن [Name] قبل يكون معنا") is mandatory, not optional
- Ending with "وصية / نصيحة" (a final piece of advice) is a strong cultural tradition for interview closings

## 2. Podcast Episode Structure

### Standard Arabic podcast episode structure
```
[Hook — 30-60 seconds]: Start in the middle of something compelling
[Introduction of host and episode framing]: "في الحلقة دي..."
[Guest introduction — personal, not just title]: Who is this person as a human?
[Origin story / background]: How did they get here?
[Core topic — main conversation]: 3-5 major beats
[Challenges / failure / lessons]: The "real" part
[Current state / project / mission]: What they're building now
[Advice / وصية]: Final wisdom
[Outro]: Next episode, CTA
```

## 3. Interview Question Bank — Arabic Contexts

### The origin question (always ask this)
```
كل ناس عندها نقطة بداية — لحظة أو قرار غيّر اتجاههم.

إيه نقطة البداية عندك؟
```

### The "real talk" opener (builds trust, signals depth)
```
قبل ما نبدأ — أنا مش عارف ناخد المحادثة دي على السطح.

إيه الحاجة اللي ما بتتكلمش عنها كتير في الإنترفيوهات اللي بتعملها؟
```

### On failure (framed for Arabic audience)
```
الناس الناجحة بتتكلم عن نجاحاتها كتير.

بس الدروس الحقيقية بتيجي من التاني. إيه أصعب لحظة مريت بيها وما رجعتش تاني؟
```

### On decision-making
```
لو رجعت لقرار بتاع [X سنة] — اللي اتخذته وقتها — هل كنت هتاخده تاني؟

ولو لأ — بتعلمك إيه ده عن نفسك؟
```

### The contrarian question (for thought leadership guests)
```
في الصناعة دي فيه رأي مسلّم بيه من معظم الناس.

وأنا حاسس إنك بتختلف مع جزء منه — بس ما بتقوله صريح.

قولنا الرأي الحقيقي.
```

### On family and culture (Arabic audiences connect deeply here)
```
[Careful for KSA: adjust based on gender and cultural norms]

العيلة بتأدي دور كبير في مسيرة الناس — بتشجّع أو بتأثر أو بتشكّل الشخصية.

العيلة أثّرت إزاي في الشخص اللي إنت عليه؟
```

### The "what nobody asks you" question
```
في سؤال بتتمنى حد يسألك إياه في الإنترفيوهات — وما حدش بيسأله؟

اسأل نفسك وأجبنا.
```

### The وصية (final advice — always close with this)
```
آخر سؤال — والناس بتتعلم منه كتير:

لو عندك كلمة واحدة أو جملة واحدة للشخص اللي عمره [X] وبيمر بـ [relevant challenge] —

إيه اللي بتقوله؟
```

## 4. Host Dialogue Frameworks

### Introducing a guest (never read a LinkedIn bio)
Instead of: "ضيفي اليوم هو [Name] اللي حصل على ..."

Use:
```
فيه ناس لما بتتعرف عليهم بتحسي إن في حاجة مختلفة في طريقة تفكيرهم.

ضيفي النهارده واحد من دول — [Name].

وأنا مش هعرّفه بـ [title] — لأن ده مش اللي بيعرّفه.

[Name] هو الشخص اللي [personal defining quality or story in one sentence].

أهلاً وسهلاً يا [Name].
```

### Transitioning between topics
```
ده كلام جميل. وخلّاني أاخد الموضوع لمكان تاني —

لأن في حاجة قالها [Name] من شوية بيت في بالي:

[Quote from earlier in the conversation]

إيه اللي وراه ده؟
```

### Challenging a guest without confrontation
```
سامحني لو السؤال ده فيه استفزاز شوية —

لكن في ناس هتسمع كلامك وهتقول: "[opposing view in one sentence]".

بتقول إيه لهم؟
```

### When a guest gives a generic answer
```
كل ده صح — وأنا مش شايف خلاف.

بس أنا عايز الإجابة اللي تحت ده. الحاجة اللي إنت عارفها وصعبة تتقال.

قولها.
```

## 5. Press Interview Prep (Spokesperson / CEO)

### Prep framework — the 3 messages
Before any press interview, define exactly 3 key messages. Every answer brings it back to one of these.

```
رسالة ١: [Core message]
رسالة ٢: [Supporting message]
رسالة ٣: [Differentiating message]
```

### Bridging back to message
When asked a question you want to redirect:
```
ده سؤال مهم — وأنا هجاوب عليه. بس خليني أضيف حاجة مهمة أول:

[State your key message]

ورجعاً للسؤال: [brief response]
```

### Handling a hostile question
```
أنا بافهم السؤال — وأنا مش هرفضه.

لكن خليني أضع الصورة الكاملة عشان ما نبقاش بنتكلم في فراغ:

[Context] + [Your position] + [Evidence]
```

### "No comment" — Arabic alternative (never say "no comment" — it sounds like hiding)
```
الموضوع ده دلوقتي في [appropriate context: legal review / board discussion / early stage].

لما بيكون عندي اللي يكمّل الصورة — هتكون الأول اللي أبلّغه.
```

## 6. Show Notes & Episode Description Templates

### Arabic podcast description (Instagram / Spotify)
```
الحلقة [Episode #] — [Guest Name]

[One-sentence hook that starts with a bold claim or provocative question]

في الحلقة دي، اتكلمنا عن:
— [Topic 1]
— [Topic 2]
— [Topic 3]

وكمان: [Unexpected or personal moment from the episode]

[Guest's Instagram / LinkedIn]

استمع / شوف دلوقتي 🎙️ [Link or "الرابط في البايو"]
```

### YouTube chapter titles (Arabic)
Chapters help Arabic audiences navigate long-form content:
```
00:00 — المقدمة وتعريف [Guest]
05:30 — بداية الرحلة وقصة النجاح
18:45 — أكبر فشل في حياتهم
30:20 — رأيهم في [Controversial topic]
48:00 — كيف [Key lesson]
57:30 — الوصية الأخيرة
```

## 7. Dialect Guide for Podcasts & Interviews

| Format | Dialect | Notes |
|---|---|---|
| Egyptian podcast (Masri guest) | Masri L3-L4 | Natural, conversational, light humor OK |
| KSA podcast | KSA L3-L4 | Measured pace, respect titles initially |
| Gulf/Khaliji podcast | Khaliji L3 | Regional warmth; cultural references |
| Levantine podcast | Levantine L3 | Intellectual, storytelling-rich |
| Pan-Arab business podcast | White Dialect L3 | Neutral; avoids alienating any market |
| Press / official interview | L4 or Business MSA | Depends on outlet formality |
| YouTube Arabic tech content | White Dialect / Masri L3 | Egypt is the largest Arabic YouTube market |

**Mixing rule:** Host can stay in their natural dialect. Guest should be allowed theirs. Don't force code-switching — it creates awkward moments and the audience handles natural dialect mixing fine.
