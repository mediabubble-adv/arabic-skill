# 🎧 Customer Service — Arabic Scripts & Flows

**Use when:** Writing customer support scripts, complaint response templates, FAQ answer flows, chatbot conversation trees, WhatsApp support templates, or escalation scripts in Arabic
**Load alongside:** Target dialect file + relevant domain file

## 1. Arabic Customer Service Culture

### The emotional register is higher
Arabic customers express frustration more expressively than Western ones. "ده مش معقول!" (this is unacceptable!) and "أنا مش راضي!" (I'm not satisfied!) are not necessarily crisis signals — they're culturally normal expressions of dissatisfaction. Don't escalate in kind. De-escalate with warmth.

### Personal address is mandatory
Never address a customer as "العميل" (the customer) or impersonally. Use their name, or اللي حضر (the gentleman/lady here). Impersonal Arabic customer service reads as dismissive.

### Apology culture
Arabic culture values sincere acknowledgment. A well-framed apology is not an admission of legal liability — it's a social signal that you see the person and take their concern seriously. Never withhold acknowledgment to "avoid blame."

### The complaint amplification problem
Dissatisfied Arabic customers will tell their entire network (family, WhatsApp groups, social media) about a bad experience much faster and more widely than Western customers. Resolving a complaint with grace often turns them into advocates for the same reason.

## 2. The 5-Step De-escalation Framework

For any complaint or negative interaction:

### Step 1: Acknowledge (not apologize yet)
```
أنا سامعك وفاهم — وده فعلاً مش اللي المفروض يحصل.
```
— Validates the emotion before addressing the problem

### Step 2: Apologize sincerely
```
آسف جداً إنك اتعرضت لده. ده مش مستوى الخدمة اللي بنعدّه لعملائنا.
```
— Specific to the situation; not generic "we apologize for any inconvenience"

### Step 3: Clarify (without making them repeat)
```
عشان أحل الموضوع بسرعة: [confirm you understood the issue]. ده صح؟
```
— Summarize their issue back to them; shows you listened

### Step 4: Action + timeline
```
هقدر أحل الموضوع ده بطريقتين:
الأولى: [Option A + timeline]
الثانية: [Option B + timeline]

إيه اللي بيناسبك أكتر؟
```
— Give them agency; choice restores sense of control after frustration

### Step 5: Confirm + follow up promise
```
تمام. هقوم بـ [action] دلوقتي. وهتلاقي [result] في [specific timeframe].

لو في أي حاجة تاني — أنا هنا.

اسمي [Name] — وأنا شخصياً متابع معاك.
```
— Personal accountability: a named person is responsible, not "the team"

## 3. WhatsApp Support Templates

### First response (under 2 minutes ideal)
```
مرحبا [Name] 👋

وصلت رسالتك — وأنا معاك.

قولي أكتر إيه اللي بيحصل عشان أساعدك أسرع.
```

### Request for order/account info
```
تمام! عشان أقدر أساعدك:

رقم طلبك أو رقم الحساب موجود معاك؟

(لو مش موجود، اسمك الكامل + رقم تليفونك كافي.)
```

### Delay acknowledgment
```
أنا شايف الموضوع [Name].

فيه تأخير مش متوقع بسبب [brief reason — never lie]. ده مش عادي بالنسبة لنا وأنا فاهم إنه محبطك.

التوقع الحالي: [new honest timeline].

لو ده مش مناسبك، عندنا [alternative] — هل بيناسبك؟
```

### Refund/return confirmation
```
تمام — تمت الموافقة على [refund/return].

هتلاقيه في [payment method] خلال [timeframe].

لو في أي سؤال تاني — أنا هنا. وشكراً على صبرك.
```

### Escalation to manager
```
[Name] — هقوم بتصعيد موضوعك لمدير الخدمة عشان يتابع معاك شخصياً.

هيتواصل معاك في [timeframe].

وأنا باعتذر تاني إن الأمور وصلت لده — ده مش اللي بتستاهله.
```

## 4. Complaint Handling by Type

### Product defect (عيب في المنتج)
```
أنا آسف جداً — ده مش مقبول.

هنعمل [exchange/refund/replacement] فوراً.

محتاج منك: صورة للمنتج + رقم الطلب.

وخليني أتأكد: عايز استبدال بنفس المنتج ولا استرداد؟
```

### Wrong order (طلب غلط)
```
آسف جداً على الخطأ ده.

هنعمل استلام للمنتج الغلط منك + توصيل الصح في [timeframe].

ومحتاج أتأكد من عنوانك: [address]. ده نفسه؟
```

### Delivery delay (تأخر في التوصيل)
```
شايف إن طلبك [order#] لسه ما وصلش.

المعلومات عندي بتقول [current status].

أنا بتابع مع فريق التوصيل دلوقتي وهرجع ليك خلال [X] دقايق بتحديث.
```

### Billing dispute (خلاف على فاتورة)
```
هفتح حالة مراجعة للفاتورة دي دلوقتي.

في الغالب بيخلص خلال [X] ساعة.

لو الخطأ من طرفنا، هيتم التصحيح فوراً + مش هتتحمل أي رسوم إضافية.
```

## 5. Chatbot / Automated Flow Scripts

### Opening greeting
```
مرحبا بك! 👋

أنا [Bot name]، مساعدك في [Company].

إزاي أقدر أساعدك النهارده؟

🔹 تتبع طلبي
🔹 سؤال عن المنتجات
🔹 مشكلة في الطلب
🔹 التكلم مع شخص
```

### Handoff to human agent
```
فاهم — الموضوع ده محتاج شخص متخصص.

بحوّلك دلوقتي لزميلنا [Name].

وقت الانتظار: [X] دقايق تقريباً.

شكراً على صبرك 🙂
```

### Out-of-hours response
```
مرحبا!

دلوقتي برا أوقات عملنا ([hours]).

لكن رسالتك وصلت — وهترجعلك أول ما نفتح في [day] الساعة [time].

لو الموضوع عاجل: [emergency contact or alternative channel]
```

## 6. FAQ Answer Format

### Structure for every FAQ answer
```
[Direct answer in first line — no preamble]

[Supporting explanation if needed — 1-2 lines max]

[Next step / action if applicable]

[Invite follow-up question]
```

### Example: "إزاي أرجع طلبي؟" (How do I return my order?)
```
الإرجاع متاح خلال [X] يوم من تاريخ الاستلام.

كيفية الإرجاع:
1. ابعتلنا صورة للمنتج على WhatsApp
2. هنرتب استلامه منك مجاناً
3. المبلغ بيرجع في [X] أيام عمل

في سؤال تاني؟
```

## 7. Dialect Guide for Customer Service

| Channel | Audience | Dialect | Notes |
|---|---|---|---|
| WhatsApp chat (Egypt B2C) | Mass market | Masri L3 | Warm, accessible |
| WhatsApp chat (KSA B2C) | Consumer | KSA L3 | Regional markers build trust |
| Live chat (pan-Arab platform) | Mixed | White Dialect L3 | Neutral, accessible |
| Phone support (any market) | All | Match caller's dialect | Mirror their language immediately |
| Email support (B2B) | Professional | L4 or Business MSA | Formal but warm |
| Chatbot (pan-Arab) | All | White Dialect, simple sentences | Clarity over style |
| Social media DMs | Varied | Match the platform's audience register | |

**Mirror rule:** In live chat and phone, immediately shift to the dialect the customer uses. Nothing builds rapport faster.

## 8. Phrases That Never Work in Arabic Customer Service

| Avoid | Why | Use instead |
|---|---|---|
| "وفقاً للسياسة" (per policy) | Shifts blame to an abstraction; feels cold | "إحنا هنعمل [X] عشانك" |
| "هذا ليس بإمكاننا فعله" | Negative framing; signals giving up | "اللي أقدر أعمله دلوقتي هو [X]" |
| "يُرجى الانتظار" (please wait) — alone | Leaves them in a void | "شايفك — دقيقة بس وأنا معاك" |
| "تم استلام شكواكم" | Robotic; impersonal | "[Name]، وصلت رسالتك وأنا شخصياً بتابع" |
| Apology without action | Feels hollow | Apology + specific next step |
| "كما ذكرت سابقاً" | Condescending | Just re-state the point |

## 9. Social Media Public Complaint Response

When a complaint is public (Instagram comment, Twitter/X mention, Facebook post):

### Formula: Acknowledge → Take it private
```
[Name]، أنا آسف إنك مريت بالتجربة دي.

ده مش اللي بنعدّه ليك.

ممكن تبعتلنا رسالة خاصة بتفاصيلك عشان نحل الموضوع فوراً؟
```

**Rules for public responses:**
- Never argue in public — even if the customer is wrong
- Never offer compensation publicly (creates entitlement precedent)
- Always move to private channel within 1-2 exchanges
- Public tone should be warmer and more empathetic than private (others are watching)
