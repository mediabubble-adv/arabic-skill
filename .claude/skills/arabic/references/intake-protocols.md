# 📋 Intake Protocols — 70/30 System

## The Rule
Never ask more than 5 questions. Apply the 70/30 split:
- **70% Static (3 questions):** Pre-set core questions for this workspace type — always asked
- **30% Dynamic (1–2 questions):** Generated by you based on the exact niche, product, country, and prompt

**Maximum questions per intake session:** 5 total (3 static + up to 2 dynamic)

**Archetype confirmation note:** For commercial tasks (ads, sales, campaigns, brand voice), consider asking "Who is this buyer at heart?" using the archetypes from `references/persuasion-arab-psychology.md` (Family Guardian, Generous Host, Faithful Steward, Ambitious Achiever, Community Pillar, Heritage Keeper). This is a **dynamic slot question** — it counts toward the 1–2 dynamic questions, not on top.

**Error Recovery — Contradiction Protocol:**
If a user's answer contradicts their initial prompt, pause and say:
"قبل ما نكمل — لاحظت إن [X] يتعارض مع [Y]. أيهما نتبع؟"
Never guess past a contradiction. Wait for clarification.

**Returning Client Protocol:**
If the user mentions a previous brand guide, brief, or prior output — ask first:
"عندك بريف أو دليل صوت علامة تجارية سابق؟ لو موجود، شاركه معايا وهنبدأ منه."
Skip questions covered by the existing brief.

## 1. 🌐 Website Owner

**Static (always ask):**
1. أنهي صفحات محتاجة كوبي؟ وأيها الأولوية؟ (الرئيسية / عن / خدمات / أسئلة شائعة / تواصل)
2. إيه أكبر شك أو خوف عند الزائر قبل ما يتخذ قرار؟
3. إيه الإجراء الوحيد اللي عايزه يعمله في آخر الصفحة؟

**Dynamic rule:** اسأل سؤال واحد عن استراتيجية عرض السعر — هل يعرضون السعر مباشرة ولا بيطلبوا التواصل أولاً؟ (يختلف حسب البلد والصناعة)

---

## 2. 🏷 Brand Builder

**Static (always ask):**
1. وصف العلامة التجارية بـ3 كلمات — ما تمثله، مش ما تبيعه
2. إيه اللي تؤمن بيه العلامة التجارية وما يؤمن بيه المنافسون؟
3. 3 علامات تجارية من خارج صناعتك بتعجبك طريقة كلامها ليه؟

**Dynamic rule:** اسأل عن موقفهم من العبارات الدينية أو الثقافية (بإذن الله، بفضل الله، الحمد لله) في محتواهم — يعني حاجة تخص سوقهم المحدد.  
**Commercial dynamic option:** For brand voice guides, consider "أي من الأنماط الثقافية أقرب للعلامة التجارية؟" to anchor voice in an archetype (per `persuasion-arab-psychology.md`).

---

## 3. 🏢 Ad Agency

**Static (always ask):**
1. إيه التوتر الثقافي أو الـInsight المحوري اللي الحملة بتمسه؟
2. إيه الـMedia Mix؟ (تلفزيون / OOH / ديجيتال / سوشيال)
3. الحملة بلهجة واحدة ولا بان-آراب مع لوكالايزيشن؟

**Dynamic rule:** اسأل عن بطل القصة — هل هو البراند، المستهلك، ولا المنتج نفسه؟ (يتحدد حسب الكاتيجوري).  
**Commercial dynamic option:** For high-stakes campaigns, consider "من هو الشاري من قلبه — حامي العيلة، الكريم، الأمين، الطموح، عمود المجتمع، ولا حارس التراث؟" (Archetype question per `persuasion-arab-psychology.md`)

---

## 4. 📈 Performance Marketer

**Static (always ask):**
1. إيه الـPain Point الرئيسي للجمهور بكلامه هو — مش كلامك انت؟
2. إيه أقوى سبب يخليه يتصرف دلوقتي؟ (عرض محدود / ندرة / ضمان)
3. البلاتفورم والفورمات إيه؟ (ميتا / ريلز / تيك توك / سناب / كاروسيل)

**Dynamic rule:** اسأل عن الـHooks أو الأفكار اللي اتجربت قبل كده وما اشتغلتش — عشان نتجنبها.

---

## 5. 💰 Sales Professional

**Static (always ask):**
1. إيه أكبر 3 اعتراضات بيقولها الناس قبل الشراء؟ (السعر / الثقة / التوقيت)
2. إيه الدليل الموجود؟ (كيس ستاديز / توصيات / أرقام / ضمان)
3. الفورمات المطلوب؟ (صفحة مبيعات / VSL / سلسلة واتساب / إيميل / بروبوزل)

**Dynamic rule:** اسأل عن قناة الإغلاق — واتساب ولا إيميل ولا تليفون؟ ده بيحدد نبرة وطول الـCTA.  
**Commercial dynamic option:** For high-value sales, consider "من هو الشاري الأساسي — بالنسبة لنفسيته واحتياجاته?" (Archetype per `persuasion-arab-psychology.md`).

## 6. 🎥 Video Creator

**Static (always ask):**
1. إيه التحول اللي المشاهد هيحسه في آخر الفيديو؟
2. الطول المستهدف بالظبط؟ (ده بيحدد الوتيرة وعدد الكلمات ومواضع الـRecap)
3. شخصية الشانل إيه؟ (رسمي / كوميدي / تعليمي / عاطفي / دوكيومنتري)

**Dynamic rule:** اسأل عن الـPattern Interrupt المطلوب في أول 3 ثواني — إيه اللي يوقف السكرول بصرياً أو صوتياً؟

---

## 7. ✍️ Blogger / SEO / AEO

**Static (always ask):**
1. إيه الكيورد أو السؤال الأساسي — وبأي لهجة أو نوع عربي؟
2. إيه الـSearch Intent؟ (معلوماتي / تجاري / تحويلي / سؤال لـAI)
3. المحتوى لجوجل التقليدي، ولا AI Search (Perplexity / ChatGPT)، ولا الاتنين؟

**Dynamic rule:** اسأل عن نقطة التمييز — إيه اللي عندهم ومش عند المنافسين؟ (داتا / تجربة / زاوية / رأي)

---

## 8. 📱 Social Creator

**Static (always ask):**
1. البلاتفورم والفورمات إيه؟ (إنستاجرام فيد / ريلز / تيك توك / X ثريد / لينكدإن / سناب / واتساب بروكاست / تليجرام / ثريدز)
2. إيه الإيموشن اللي المتابع يحسه في آخر البوست؟ (مُلهَم / مُسلَّي / مُفيد / مصدوم)
3. إيه الـCTA؟ (فولو / سيف / شير / لينك / دم / كومنت)

**Dynamic rule:** هل ده أكاونت فيه تاريخ (نعرف الأداء السابق) ولا أكاونت جديد؟ لو جديد، مين الجمهور المستهدف بالظبط؟

---

## 9. 📚 Author / Novelist

**Static (always ask):**
1. إيه الجانر والـPremise والمسار؟ (فيكشن / نون فيكشن / سيلف هيلب / بيزنس / تاريخي)
2. إحنا فين في الرحلة؟ (أوتلاين / مسودة فصل / مخطوطة كاملة / تخطيط سيريز)
3. إيه الوعد العاطفي للكتاب — إيه الإحساس اللي القارئ بياخده بعد الصفحة الأخيرة؟

**Dynamic rule:** اسأل عن رجيستر اللغة — الراوي بيكلم بنفس لهجة الشخصيات ولا رجيستر مختلف؟ ده بيحكم معمار اللغة كله.

---

## 10. 💻 Indie Dev / SaaS

**Static (always ask):**
1. أنهي شاشة أو فلو محتاج كوبي؟ (سبلاش / أونبوردنج / پيوول / Empty State / إيرور / سيتينجز)
2. فيه قيود محددة في عدد الحروف لأي عنصر UI؟ (أزرار / ليبلز / ألرتس / تولتيبس)
3. إيه الإجراء الوحيد اللي المستخدم يعمله في الشاشة دي؟

**Dynamic rule:** اسأل عن قيود الـRTL — فيه عناصر لازم تتصرف مختلف في النسخة العربية عن الإنجليزية في اللياوت أو الطول؟

## 11. 📝 Professional Document

**Static (always ask):**
1. إيه نوع الوثيقة؟ (عقد / سكيل AI / تعليمات وكيل / نص امتثال / قواعد)
2. مين الأطراف أو الجمهور؟ (أطراف العقد / مستخدمو السكيل / جمهور آخر)
3. أنهي بلد أو معيار قانوني بيحكم الوثيقة دي؟

**Dynamic rule:** اسأل عن مستوى الرسمية وهل محتاجين نسخة ثنائية اللغة (عربي + إنجليزي)، وأيهما يسود في حالة التعارض.

---

## 12. 🤖 Skill / Agent / Subagent Writing

**Static (always ask):**
1. إيه اللي السكيل أو الوكيل ده المفروض يعمله — إيه وظيفته الأساسية؟
2. امتى يُفعَّل — إيه العبارات أو السياقات اللي المفروض تشغله؟
3. إيه الفورمات المتوقع للـOutput وتقريباً كام طوله؟

**Dynamic rule:** اسأل عن البيئة اللي هيشتغل فيها (Claude Code / Claude.ai / API / غيرها) وهل محتاج عبارات تفعيل بالعربي.
