# `.arabic/` workspace

Local state for Awesome Arabic Skill — safe to commit or gitignore per team policy.

| Path | Purpose |
|------|---------|
| `config.yaml` | Default dialect, market, register |
| `briefs/` | YAML briefs for `--brief` |
| `projects/` | Plan mode trees (`plan.md`, `output/`) |
| `voice.md` | Created by `/arabic voice save` |
| `audits/` | Optional audit snapshots |

First commands: `/arabic guide` · `/arabic write caption --brief briefs/example.yaml`
