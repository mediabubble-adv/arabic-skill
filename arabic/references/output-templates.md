# Output Templates

Select the correct template based on content type before generating output.

---

## Template A — Short-form Creative
**Use for:** Taglines, slogans, captions, CTAs, ad hooks, one-liners (12 variants in 3 tiers of 4)

**Format note for Instagram:** If the request is for captions, produce separate sets for:
- **Feed captions** (regular posts/carousels) — longer, can include storytelling
- **Reels captions** (max 2200 chars, but first 125 chars must hook before "More") — punchy, drives views
Produce both unless the user specifies only one format.

```
🌍 اللهجة / Dialect: [Dialect]  |  🎯 Goal: [Objective]  |  📱 Platform: [If applicable]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

TIER A — Bold / Direct
١. [Arabic text] → [English meaning, not translation] | Lever: [psychological trigger]
٢. [Arabic text] → [English meaning] | Lever: [psychological trigger]
٣. [Arabic text] → [English meaning] | Lever: [psychological trigger]
٤. [Arabic text] → [English meaning] | Lever: [psychological trigger]

TIER B — Emotional / Story
٥. [Arabic text] → [English meaning] | Lever: [psychological trigger]
٦. [Arabic text] → [English meaning] | Lever: [psychological trigger]
٧. [Arabic text] → [English meaning] | Lever: [psychological trigger]
٨. [Arabic text] → [English meaning] | Lever: [psychological trigger]

TIER C — Playful / Unexpected
٩. [Arabic text] → [English meaning] | Lever: [psychological trigger]
١٠. [Arabic text] → [English meaning] | Lever: [psychological trigger]
١١. [Arabic text] → [English meaning] | Lever: [psychological trigger]
١٢. [Arabic text] → [English meaning] | Lever: [psychological trigger]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🧠 Cultural Rationale:
[1–2 sentences on why these directions work for this dialect/region]

📐 Specs:
[Character counts / Hashtag count and strategy / Platform format note (Feed vs. Reels if Instagram)]
```

## Template B — Long-form Persuasive
**Use for:** Blog posts, landing pages, sales pages, website sections, email sequences

```
🌍 اللهجة / Dialect: [Dialect]  |  🏢 Workspace: [Industry]  |  🎯 Goal: [Objective]

🤖 AEO Mode (activate when goal includes AI search — Perplexity / ChatGPT / Google AI Overviews):
Structure as: Paragraph 1 = direct 1–2 sentence answer to the H1 question.
Body = SEO development of that answer. Final section = FAQ (min 5 entries: Q → Direct Answer → Expansion).
H1 = the exact question the audience types into Perplexity or ChatGPT.
Use tables and numbered lists wherever comparison or process appears — AI engines extract these directly.
```

## Template D — Brand Asset
**Use for:** Brand voice guide, brand story, manifesto, style guide
**Target document length:** 1500–2500 words in Arabic. Below 1500 is too thin for a real brand document. Above 2500 risks bloat that teams won't read. Aim for 2000 words as the quality benchmark.

```
🏷 Brand: [Brand Name]  |  🌍 Primary Dialect: [Dialect]  |  🏢 Industry: [Category]
```

## Template F — Professional Document

```

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📐 Document Plan (before full draft):
- Purpose: [One sentence]
- Governing framework: [Country law / AI platform / Internal standard]
- Required sections: [List all mandatory sections]
- Formality level: [Legal formal / Professional / Semi-formal]
- Special constraints: [Word limits / Format requirements / Language hierarchy in bilingual docs]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📝 FOR CONTRACTS — Full Arabic Document:

# [Document Title — formal Arabic, e.g., عقد تقديم خدمات]

## ديباجة العقد / Preamble
[Opening clause — purpose and scope of the document]

[Clause text]

### [Continue as needed — البند الثالث، الرابع، إلخ]

## الإقرارات والتوقيعات / Declarations and Signatures
[Final declarations, signature blocks with date/place fields]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🧠 Document QA:
```
