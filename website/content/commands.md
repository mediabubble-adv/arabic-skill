# سطح الأوامر

أمر واحد — `/arabic` — وتحته مسار العمل بالكامل.

## الأوامر الأساسية

| الأمر | الشرح |
|-------|--------|
| `/arabic guide` | الفكرة مش واضحة؟ ابدأ هنا — يوضح، يسأل، ويوصي |
| `/arabic write meta --dialect masri` | إعلان ميتا بالمصري من برّيف جاهز |
| `/arabic audit --file content.md` | مراجعة نص عربي قائم مع تشخيص واضح |
| `/arabic plan website` | موقع متعدد الصفحات — خطة أولاً، تنفيذ بعد الموافقة |
| `/arabic auto explain` | مسح المشروع وشرحه بعربي مباشر |
| `/arabic research meta-ads` | جمع مصادر قبل الكتابة |
| `/arabic coach` | تحسين البرومبت قبل التنفيذ |
| `/arabic voice save` | حفظ صوت البراند لنغمة ثابتة في كل كتابة |

## أوضاع الشغل

| الوضع | متى تستخدمه |
|-------|-------------|
| استشارة | الفكرة لسه مش محسومة |
| Pro | عندك برّيف كامل وعايز تكتب مباشرة |
| مشروع | موقع، حملة، أو كتاب — خطة ثم تنفيذ مرحلي |
| تدقيق | نص عربي محتاج مراجعة قبل النشر |
| Coach | البرومبت ضعيف ومحتاج تصليح |

## ابدأ

**ابدأ بـ `/arabic guide`** — ثم **ثبّت المهارة** إن لم تكن مركّبة بعد.
