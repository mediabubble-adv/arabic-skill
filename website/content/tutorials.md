# اتعلّم بالعربي

دروس من شغل المهارة نفسها — **مش مجرد ترجمة** من إنجليزي.

## الدرس ١: ثبّت المهارة في ٣ دقايق

**النتيجة:** G14 منسوخ + `/arabic guide` شغال.

1. افتح التيرمنال في مشروعك
2. انسخ: `npx @mediabubble-adv/arabic-skill install --target cursor`
3. اكتب `/arabic guide` في Cursor

**~٣ دقايق · مبتدئ**

## الدرس ٢: اكتب أول بوست مصري

**النتيجة:** منشور سوشيال مصري من برّيف بسيط.

1. اكتب الفكرة بشكل جزئي — مش لازم برّيف كامل
2. `/arabic guide` هيوضّح ويسأل
3. `/arabic write` يولّد المحتوى
4. `/arabic audit` يراجع قبل التسليم

**~١٠ دقايق · مبتدئ**

## الدرس ٣: راجع نصك قبل ما تنشر

**النتيجة:** تقرير تدقيق + قائمة إصلاحات.

1. حط النص في ملف `.md`
2. `/arabic audit --file path/to/file.md`
3. طبّق أهم إصلاح — واطلب إعادة تدقيق لو محتاج

**~٥ دقايق · متوسط**

## مسار كامل (معاينة)

استشارة → توضيح → توصية → كتابة → مراجعة

**ثبّت المهارة** — وابدأ الدرس الأول.
