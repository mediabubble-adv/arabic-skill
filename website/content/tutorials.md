# اتعلّم بالعربي

دروس من شغل المهارة نفسها — **مش مجرد ترجمة** من إنجليزي.

## الدرس ١: ثبّت المهارة في ٣ دقايق

**النتيجة:** أمر G14 منسوخ + `/arabic guide` شغال.

1. افتح التيرمنال في مشروعك
2. انسخ: `npx @mediabubble-adv/arabic-skill@latest install --target cursor`
3. اكتب `/arabic guide` في Cursor

**~٣ دقايق · مبتدئ**

## الدرس ٢: اكتب أول منشور مصري

**النتيجة:** منشور سوشيال مصري من برّيف بسيط.

1. اكتب الفكرة بشكل جزئي — مش شرط برّيف كامل
2. `/arabic guide` يوضّح ويسأل
3. `/arabic write` يولّد المحتوى
4. `/arabic audit` يراجع قبل التسليم

**~١٠ دقايق · مبتدئ**

## الدرس ٣: راجع النص قبل النشر

**النتيجة:** تقرير تدقيق + قائمة إصلاحات.

1. ضع النص في ملف `.md`
2. `/arabic audit --file path/to/file.md`
3. طبّق أهم إصلاح — واطلب إعادة تدقيق عند الحاجة

**~٥ دقايق · متوسط**

## المسار الكامل (معاينة)

استشارة → توضيح → توصية → كتابة → مراجعة

**ثبّت المهارة** — وابدأ بالدرس الأول.

الموقع كله اتكتب بـ `/arabic` — [مسار البناء](/about).
