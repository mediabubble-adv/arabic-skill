# Product

## Register

brand

## Users

Arabic content creators, marketers, and developers evaluating AI writing tools — often at night on a laptop, comparing install friction and output quality before committing. Primary segments: Cursor/Claude/Codex power users (want G14 copy-ready install), founders new to "skills" (need trust + examples), and technical evaluators (commands, docs, GitHub).

## Product Purpose

Marketing site for **Awesome Arabic Skill** (`arabic`): drive install conversion (`npx @mediabubble-adv/arabic-skill install`) while proving the skill through dogfood transparency (Masri copy, `/about` audit trail, footer SSOT). Success = confident install copy + credible Arabic-first positioning (**مش مجرد ترجمة**).

## Brand Personality

Confident, editorial, Egyptian-first, dev-credible. Voice is مصري طبيعي in all page chrome — not formal MSA marketing, not translated English UI. The install command is a designed artifact, not an afterthought.

Three words: **مصري · شريك · شغل حقيقي**

## Anti-references

- Generic AI SaaS (purple gradients, gradient text, hero-metric template, identical 3-card feature rows)
- English chrome on Masri routes
- Editorial-magazine Klim clone (italic display serif + ruled separators + no imagery) without magazine reason
- "Skills marketplace" template aesthetics
- Side-stripe accent borders, glassmorphism-as-default, modal-first patterns
- Bare translation positioning ("مش ترجمة" without **مجرد**)

## Design Principles

1. **Install-first funnel** — every route supports copy-ready path to `/install` and G14 block.
2. **Show, don't tell** — before/after, mode flow, and dogfood trail beat adjectives.
3. **Arabic-first RTL** — layout, copy, and hierarchy built for RTL; LTR only for code blocks and email fields.
4. **Practice what you preach** — site copy produced by `/arabic`; transparency on `/about` and footer.
5. **Committed, not timid** — brand teal carries weight; dark-first scene fits focused nighttime evaluation.

## Accessibility & Inclusion

WCAG AA minimum (4.5:1 body, 3:1 large text). Skip link, semantic landmarks, ARIA tablists on mobile toggles. `prefers-reduced-motion` respected. RTL-native reading order; keyboard paths on all interactive chrome.
